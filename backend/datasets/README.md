# Uploaded datasets

Place any `.csv` file in this folder. Each file becomes one table in the Sandbox.

- File `Book1.csv` → table `book1`
- File `indian-state-level-election.csv` → table `indian_state_level_election`

The first row must be column headers. Large files load the first 500 rows automatically.

After adding files, redeploy the backend on Render.
